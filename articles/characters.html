Search Results for

    Characters

    Disney characters such as Mickey Mouse, Donald Duck, and so on.

    Character schema

    Key Type Description
    id uuid The id of the character.
    name string The name of the character.
    age int The age of the character.
    weight float The weight of the character
    story string The story of the character
    imageUrl string Link to the character's image
    moviesOrSeries array The movies or series related to the character

    Route Parameters

    Key Type Description
    characterId uuid The id of the character
    movieId uuid The id of the movie

    Query Parameters

    Key Type Description
    PageNumber int Pagination
    PageSize int Pagination
    Age int Filter by the age of the character
    Weight float Filter by the weight of the character
    MoviesOrSeries string Filter by the movie or series
    Name string Search by name

    Request:

    [DELETE] A character
