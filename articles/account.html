    Account schema

    Key Type Description
    userName string User name
    email string The user's email
    password string The user's password
    role int The user's role

    [POST] Register a user

    Request:

    https://disney-api.app.csharpjourney.com/auth/register

    Body

    {
  "userName": "string",
  "email": "string",
  "name": "string",
  "password": "string",
  "role": 0
}

    Response:

    200 Success

    Request:

}
}

    Response:

    200 Success
